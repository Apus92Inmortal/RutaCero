# Assets oficiales de marca - Ruta Cero

Esta carpeta contiene los logos oficiales de Ruta Cero by INTRA.

Las versiones activas son PNG con fondo transparente real, preparadas para usarse sobre superficies claras,
oscuras o de color dentro de la app.

## Archivos

- ruta-cero-logo-primary.png  
  Uso: landing, header principal, documentacion, piezas comerciales y pantallas donde haya espacio suficiente.

- ruta-cero-logo-compact.png  
  Uso: navbar, mobile header, tarjetas, espacios horizontales reducidos.

- ruta-cero-logo-monochrome.png  
  Uso: documentos, fondos claros, impresion, estados donde se requiera una sola tinta.

- ruta-cero-isotipo-app-icon.png  
  Uso: favicon, icono de app, avatar, PWA, accesos directos, splash o lugares donde no quepa el wordmark completo.

## Reglas de uso

- No deformar los logos.
- No cambiar los colores oficiales.
- No separar "Ruta Cero" de la arquitectura "by INTRA" en piezas principales.
- Usar el isotipo solo cuando el espacio sea reducido.
- Mantener fondo limpio y suficiente aire visual.
- Usar azul profundo #0B2C4A y verde progreso #2ECC71 como colores oficiales principales.
- No usar versiones antiguas o experimentales en produccion.
- No reemplazar estos archivos por exportaciones con cuadriculado o fondo horneado.
